SensNodeclose

The SensNode device is the core of the Sensaura framework - each SenseNode consists of a ARM based CPU board and one or more sensor backpacks that measure the real world.

A standard form factor and an open source design allows for a wide range of processors and sensors to be used.

SensNetclose

The SensNet protocol allows multiple SensNode devices to talk to a single hub. The protocol supports an attribute based system with automatic discovery, read and write operations as well as event generation.

The project provides a number of client libraries that allow SensNodes to be integrated into your own applications or sensor management services.

SensHubclose

The SensHub provides a rule engine and sensor management on top of the MQTT protocol. This allows data from SensNode devices, the web and other sources to be gathered in a single place.

Unlike similar systems the SensHub is a purely local implementation, no connection to the internet is required for operation and any information you generate is only sent to the internet under your control.
