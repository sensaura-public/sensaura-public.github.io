xmc1100.h

Low level all-in-one header file for the Infineon XMC1100.

Defines system parameters and special function registers for the XMC1100 to allow development without CMSIS or vendor specific libraries. Register names are as described in Reference Manual. In the case of peripheral registers, the peripheral name is pre-pended e.g. WDT_ID is used rather than ID which might conflict with another peripheral. Original version by Frank Duignan which can be found here .
